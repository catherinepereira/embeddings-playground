# Soundscape

Interactive visualization of 25,000 FMA tracks encoded as CLAP audio embeddings, then reduced with UMAP dimensionality reduction. Click any point to stream the track, toggle between 2D and 3D views of the embeddings, and filter by genre to view the distribution of tracks.

## Setup

```bash
npm install
npm run dev
```

## Data

`public/viz.json` is generated by the [soundscape-data-preprocessing](https://github.com/catherinepereira/soundscape-data-preprocessing) pipeline.
